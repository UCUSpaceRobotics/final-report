Based on competition rules, we defined the localization requirement as an absolute pose error below \qty{0.8}{\meter} over a \qty{150}{\meter} route. To satisfy this in variable lighting and feature-poor environments, we implemented the dual-layered pipeline illustrated in \autoref{fig:localization_pipeline}

\begin{figure}[!htbp]
    \centering
    \includegraphics[width=1.0\linewidth]{teams/navigation/figures/localization_pipeline}
    \caption{Localization pipeline}\label{fig:localization_pipeline}
\end{figure}

The local Extended Kalman Filter (EKF) fuses visual-inertial and wheel data for smooth, high-frequency odometry. Global SLAM corrects drift via scan matching without inducing local state jumps. For absolute positioning, the system detects known ArUco markers; inverting their 6D pose relative to \texttt{base\_link} calculates a zero-drift global pose to correct the map transform.
